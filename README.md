# Blog

### Configuration Backend
```.sh
$ cd blog_project/
$ nano config/django/.env
```
| Key    | Description   |
| :---         |     :---      |
| `DJANGO_SECRET_KEY`  | Secret key from Django  |
| `DJANGO_DEBUG`  | Debug mode True or False  |

### Run the project
```.sh
$ docker-compose up
```
